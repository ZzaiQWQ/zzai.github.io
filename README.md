欢迎来到我的个人主页仓库！这是一个基于 **HTML5 Canvas** 和 **Tailwind CSS** 构建的响应式个人主页，主打可爱像素风与沉浸式交互体验。

---

## 🎨 项目特色 (Features)

### 1. 🐱 手绘像素动画引擎
摒弃了静态图片，使用原生 **JavaScript Canvas API** 实时绘制：
* **猫咪家族**：大猫带小猫在屏幕散步，每只都有独特配色逻辑。
* **温馨工位**：包含冒热气的咖啡、窗外流云和像素书架，营造“云办公”氛围。
* **装饰元素**：左侧灵动小兔与背景星空粒子。

### 2. ✨ 趣味交互系统
* **粒子特效**：点击屏幕任意位置，迸发爱心与星星。
* **对话气泡**：
    * **PC端**：悬停触发 "Hello!"、"想吃小鱼干" 等随机台词。
    * **移动端**：优化触控逻辑，兼顾滑动防误触与点击反馈。

### 3. 🌓 丝滑日夜模式
内置 **Sakura Pink (昼)** 与 **Deep Blue (夜)** 主题。背景渐变、毛玻璃透明度及 Canvas 动画配色随开关平滑过渡。

## 🛠️ 技术栈 (Tech Stack)

| 核心组件 | 技术方案 |
| :--- | :--- |

## 🚀 快速部署 (Quick Start)

只需 3 步，即可拥有你的像素小窝：

1.  **Fork 本仓库** 或下载 `index.html`。
2.  **个性化修改**：
    * 全局搜索 `Zzai` 替换为你的昵称。
    * 修改 `blog.zaizai520.cn` 等链接为你的地址。
    * 替换 QQ 头像接口中的号码：`nk=你的QQ号`。
3.  **开启 GitHub Pages**：
    * `Settings` -> `Pages` -> 选择 `main` 分支 -> `Save`。
